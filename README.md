## Running Locally

Clone the repository

```bash
git clone https://github.com/RishiDixit-7404/rishi-dixit.git
cd rishi-dixit
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
